# FSG 虚拟机指令集设计

> 蜉熵阁 - 灵镜(ling)原生脚本执行系统  
> 版本: 1.0 | 指令数: 20条

---

## 设计原则

1. **极简性**: 控制在20条核心指令以内
2. **可扩展性**: 预留操作码空间便于未来扩展
3. **实用性**: 覆盖数据搬移、算术运算、控制流、I/O等基本场景
4. **ling优先**: 指令设计便于ling生成，目标代码简洁

---

## 寄存器模型

| 寄存器 | 用途 | 说明 |
|--------|------|------|
| R0-R7 | 通用寄存器 | 8个32位通用寄存器 |
| PC | 程序计数器 | 指向下一条指令 |
| SP | 栈指针 | 指向栈顶（向下增长） |
| BP | 基址指针 | 栈帧基址 |
| ZF | 零标志 | 上一运算结果为0时置位 |
| SF | 符号标志 | 上一运算结果为负时置位 |
| CF | 进位标志 | 溢出时置位 |

---

## 指令编码格式

### 指令结构
```
[opcode: 1 byte][operands: 0-3 bytes]
```

- 指令长度 = 1 + 操作数总字节数
- 操作数可以是：寄存器(1字节)、立即数(1/4字节)、内存地址(4字节)

### 操作码分配

| 范围 | 用途 |
|------|------|
| 0x00-0x0F | 保留/系统指令 |
| 0x10-0x1F | 数据传输指令 |
| 0x20-0x2F | 算术运算指令 |
| 0x30-0x3F | 逻辑运算指令 |
| 0x40-0x4F | 控制流指令 |
| 0x50-0x5F | I/O指令 |
| 0x60-0x6F | 扩展指令 |
| 0xF0-0xFF | 原生函数调用 |

## 核心指令详解

### 一、数据传输指令 (0x10-0x1F)

#### 1. LOAD - 从内存加载到寄存器
- **操作码**: 0x10
- **参数**: `[reg, mem_addr]` (1 + 4 字节)
- **功能**: `reg = memory[addr]`
- **示例**:
  ```
  LOAD R1, [0x100]      ; R1 = mem[0x100]
  LOAD R2, [R0]         ; R2 = mem[R0] (间接寻址)
  ```

#### 2. STORE - 从寄存器存储到内存
- **操作码**: 0x11
- **参数**: `[mem_addr, reg]` (4 + 1 字节)
- **功能**: `memory[addr] = reg`
- **示例**:
  ```
  STORE [0x100], R1     ; mem[0x100] = R1
  STORE [R0], R2        ; mem[R0] = R2 (间接寻址)
  ```

#### 3. LOADIMM - 加载立即数
- **操作码**: 0x12
- **参数**: `[reg, imm]` (1 + 4 字节)
- **功能**: `reg = imm`
- **示例**:
  ```
  LOADIMM R1, 42        ; R1 = 42
  LOADIMM R0, -1        ; R0 = -1
  ```

#### 4. PUSH - 压栈
- **操作码**: 0x13
- **参数**: `[reg]` (1 字节)
- **功能**: `SP -= 4; memory[SP] = reg`
- **示例**:
  ```
  PUSH R1               ; 将R1压栈
  PUSH R2
  ```

#### 5. POP - 弹栈
- **操作码**: 0x14
- **参数**: `[reg]` (1 字节)
- **功能**: `reg = memory[SP]; SP += 4`
- **示例**:
  ```
  POP R1                ; 弹出栈顶到R1
  ```

#### 6. MOV - 寄存器间移动
- **操作码**: 0x15
- **参数**: `[dest_reg, src_reg]` (2 字节)
- **功能**: `dest = src`
- **示例**:
  ```
  MOV R2, R1            ; R2 = R1
  ```

---

### 二、算术运算指令 (0x20-0x2F)

#### 7. ADD - 加法
- **操作码**: 0x20
- **参数**: `[dest, src1, src2]` 或 `[dest, src, imm]`
- **功能**: `dest = src1 + src2`; 影响 ZF, SF, CF
- **示例**:
  ```
  ADD R2, R1, R0        ; R2 = R1 + R0
  ADD R1, R1, 1         ; R1++
  ```

#### 8. SUB - 减法
- **操作码**: 0x21
- **参数**: `[dest, src1, src2]`
- **功能**: `dest = src1 - src2`; 影响 ZF, SF, CF
- **示例**:
  ```
  SUB R2, R1, R0        ; R2 = R1 - R0
  ```

#### 9. MUL - 乘法
- **操作码**: 0x22
- **参数**: `[dest, src1, src2]`
- **功能**: `dest = src1 * src2`; 影响 ZF, SF
- **示例**:
  ```
  MUL R2, R1, R0        ; R2 = R1 * R0
  ```

#### 10. DIV - 除法
- **操作码**: 0x23
- **参数**: `[dest, src1, src2]`
- **功能**: `dest = src1 / src2` (整数除法); 影响 ZF
- **异常**: 除零时设置CF，跳转到预设处理地址
- **示例**:
  ```
  DIV R2, R1, R0        ; R2 = R1 / R0
  ```

#### 11. NEG - 取反
- **操作码**: 0x24
- **参数**: `[reg]` (1 字节)
- **功能**: `reg = -reg`
- **示例**:
  ```
  NEG R1                ; R1 = -R1
  ```

#### 12. MOD - 取模
- **操作码**: 0x25
- **参数**: `[dest, src1, src2]`
- **功能**: `dest = src1 % src2`
- **示例**:
  ```
  MOD R2, R1, R0        ; R2 = R1 % R0
  ```

---

### 三、比较与逻辑指令 (0x30-0x3F)

#### 13. CMP - 比较
- **操作码**: 0x30
- **参数**: `[reg1, reg2]`
- **功能**: 执行 `reg1 - reg2`，只影响标志位
- **示例**:
  ```
  CMP R1, R2            ; 比较 R1 和 R2
  ```

#### 14. AND - 按位与
- **操作码**: 0x31
- **参数**: `[dest, src1, src2]`
- **功能**: `dest = src1 & src2`
- **示例**:
  ```
  AND R2, R1, R0        ; R2 = R1 & R0
  ```

#### 15. OR - 按位或
- **操作码**: 0x32
- **参数**: `[dest, src1, src2]`
- **功能**: `dest = src1 | src2`
- **示例**:
  ```
  OR R2, R1, R0         ; R2 = R1 | R0
  ```

#### 16. XOR - 按位异或
- **操作码**: 0x33
- **参数**: `[dest, src1, src2]`
- **功能**: `dest = src1 ^ src2`
- **示例**:
  ```
  XOR R2, R1, R0        ; R2 = R1 ^ R0
  ```

#### 17. NOT - 按位取反
- **操作码**: 0x34
- **参数**: `[reg]`
- **功能**: `reg = ~reg`
- **示例**:
  ```
  NOT R1                ; R1 = ~R1
  ```

#### 18. SHL - 左移
- **操作码**: 0x35
- **参数**: `[dest, src, count]`
- **功能**: `dest = src << count`
- **示例**:
  ```
  SHL R2, R1, 2         ; R2 = R1 << 2
  ```

#### 19. SHR - 右移
- **操作码**: 0x36
- **参数**: `[dest, src, count]`
- **功能**: `dest = src >> count`
- **示例**:
  ```
  SHR R2, R1, 2         ; R2 = R1 >> 2
  ```

---

### 四、控制流指令 (0x40-0x4F)

#### 20. JMP - 无条件跳转
- **操作码**: 0x40
- **参数**: `[label_or_addr]` (4 字节)
- **功能**: `PC = addr`
- **示例**:
  ```
  JMP loop_start        ; 跳转到标签
  JMP 0x200             ; 跳转到地址
  ```

#### 21. JE / JZ - 相等/为零跳转
- **操作码**: 0x41
- **参数**: `[label_or_addr]` (4 字节)
- **条件**: ZF = 1 (上一比较相等或结果为零)
- **示例**:
  ```
  JE equal_label        ; ZF=1时跳转
  JZ loop_end           ; 结果为零时跳转
  ```

#### 22. JNE / JNZ - 不等/非零跳转
- **操作码**: 0x42
- **参数**: `[label_or_addr]` (4 字节)
- **条件**: ZF = 0
- **示例**:
  ```
  JNE not_equal         ; ZF=0时跳转
  JNZ continue          ; 非零时跳转
  ```

#### 23. JG / JGE - 大于/大于等于跳转 (有符号)
- **操作码**: 0x43 / 0x44
- **条件**: SF = OF 且 ZF = 0 (JG) / SF = OF (JGE)
- **示例**:
  ```
  JG greater_label      ; R1 > R2 时跳转
  JGE greater_eq_label  ; R1 >= R2 时跳转
  ```

#### 24. JL / JLE - 小于/小于等于跳转 (有符号)
- **操作码**: 0x45 / 0x46
- **条件**: SF ≠ OF 且 ZF = 0 (JL) / SF ≠ OF (JLE)
- **示例**:
  ```
  JL less_label         ; R1 < R2 时跳转
  JLE less_eq_label     ; R1 <= R2 时跳转
  ```

#### 25. CALL - 函数调用
- **操作码**: 0x47
- **参数**: `[label_or_addr]` (4 字节)
- **功能**: `PUSH PC; PC = addr`
- **示例**:
  ```
  CALL my_function      ; 调用函数
  ```

#### 26. RET - 函数返回
- **操作码**: 0x48
- **参数**: 无
- **功能**: `POP PC`
- **示例**:
  ```
  RET                   ; 返回调用者
  ```

#### 27. INT - 中断/系统调用
- **操作码**: 0x49
- **参数**: `[interrupt_num]` (1 字节)
- **功能**: 触发软中断，执行系统服务
- **示例**:
  ```
  INT 0x80              ; 调用系统服务
  ```

---

### 五、I/O指令 (0x50-0x5F)

#### 28. PRINT - 输出
- **操作码**: 0x50
- **参数**: `[reg]` 或 `[imm]` (1 或 4 字节)
- **功能**: 打印寄存器值或立即数到标准输出
- **示例**:
  ```
  PRINT R1              ; 打印R1的值
  PRINT 10              ; 打印换行
  PRINT "Hello"         ; 打印字符串(通过内存地址)
  ```

#### 29. INPUT - 输入
- **操作码**: 0x51
- **参数**: `[reg]` (1 字节)
- **功能**: 从标准输入读取整数到寄存器
- **示例**:
  ```
  INPUT R1              ; 读取整数到R1
  ```

#### 30. PRINTS - 输出字符串
- **操作码**: 0x52
- **参数**: `[str_addr]` (4 字节)
- **功能**: 从指定内存地址输出字符串
- **示例**:
  ```
  PRINTS str_hello      ; 输出字符串
  ```

---

### 六、特殊指令 (0x00-0x0F, 0xF0-0xFF)

#### 31. HALT - 终止程序
- **操作码**: 0x00
- **参数**: 无
- **功能**: 停止虚拟机执行
- **示例**:
  ```
  HALT                  ; 结束程序
  ```

#### 32. NOP - 空操作
- **操作码**: 0x01
- **参数**: 无
- **功能**: 占位，不执行任何操作
- **示例**:
  ```
  NOP                   ; 延时或对齐
  ```

#### 33. SYSCALL - 原生函数调用
- **操作码**: 0xF0
- **参数**: `[func_id]` (1 字节)
- **功能**: 调用注册的原生函数
- **示例**:
  ```
  SYSCALL 0x01          ; 调用func_id=1的原生函数
  ```

#### 34. DEBUG - 调试信息
- **操作码**: 0xF1
- **参数**: `[level]` (1 字节)
- **功能**: 输出调试信息
- **示例**:
  ```
  DEBUG 1               ; 输出VM状态
  ```
